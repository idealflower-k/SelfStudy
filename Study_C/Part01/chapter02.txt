02-1 Hello Wold! 들여다보기

함수
    - 적절한 입력과 그에 따른 출력이 존재하는 것

함수의 정의
    - 만들어진 함수, 실행이 가능한 함수를 일컬음
함수의 호출
    - 함수의 실행을 명령하는 행위
인자의 전달
    - 함수의 실행을 명령할 때 전달하는 입력 값

Part01/Hello.c
    - 프로그램이 실행되면 main이라는 함수를 찾아써 호출한다.
    - int : 출력 형태, 반환형(return type) / main : 함수이름 / (void) : 입력형태
    - 출력의 형태가 int이고 입력의 형태가 void인 main이라는 이름의 함수
    - { } : 함수의 몸체 (body)
    - ;(세미콜론) : 문장의 끝에 세미콜론을 사용한다.

표준함수
    - 기본적으로 제공되는 함수
표준 라이브러리
    - 표준함수들의 모임

헤더파일 선언의 필요성
    - stdio.h 라는 확장자가 .h 로 끝나는 헤더파일을 포함하라는 의미의 선언
    - "헤더파일 stdio.h에는 printf 함수의 호출에 필요한 정보가 존재한다. 따라서 이 파일의 정보를 포함하는 헤더파일 선언문이 삽입되어야 한다."
    - 헤더파일의 선언은 소스파일 맨 앞부분, main 함수 정의 이전에 와야 한다.

return 문
    - 함수를 호출한 영역으로 값을 전달(반환)
    - 현재 실행중인 함수의 종료
