vim
	일반모드
	- esc : 일반모드로 이동

		커서 이동
	- 방향키 : 커서이동
	- h, j, k, l : 왼, 아래, 위, 오른 커서이동, 숫자 + 키입력 (여러칸 이동)
	- 컨트롤 + g : 커서위치 정보 보기
	- 0 : 줄의 첫 번째 칸으로 커서이동
	- $ : 줄의 맨 마지막으로 커서이동
	- gg : 파일의 맨처음으로 커서이동
	- GG : 파일의 맨마지막으로 커서이동
	- G : 파일의 맨 마지막 줄 첫 번째 글자로 이동

		텍스트 삭제
	- x : 텍스트 한 칸씩 삭제
	- 숫자 + x : 여러 칸 삭제
	- dd : 문장 삭제
	- 숫자 + dd : 여러 문장 삭제
	- u : undo, 이전 작업 취소
	

	입력모드
	- i : 입력모드
	- a : append(추가), 커서를 한 칸 뒤로 옮기고 입력 모드로 전환
	- I : 커서를 첫 번째 칸으로 옮기고 입력 모드로 전환
	- A : 커서를 맨 마지막 칸으로 옮기고 입력 모드로 전환
	- o : open(개행), 커서를 다음줄로 옮기고 입력 모드로 전환
	- O :  커서 위에 빈 줄이 생기고 입력 모드로 전환

	명령모드
	- :w : write(저장), :w 파일명 저장 가능
	- :q : vim 종료
	- :wq : 저장하고 vim 종료
	- :q! : 경고를 무시하고 강제종료, 저장하지 않고 종료
	- /텍스트 : 텍스트검색 명령모드로 전환, n 다음 검색 내용으로 이동, N 이전 검색 내용으로 이동
	- :s/텍스트/대체 : subtitute(대체), 텍스트 치환, :%/텍스트/대체 (각 줄의 첫 번째 텍스트 치환), :%s/텍스트/대		체/g (모든 텍스트 치환), :%s/텍스트/대체/gc (확인하면서 치환)
	
	비주얼 모드
	- v : 비주얼 모드 이동
	- 방향키 : 블록 지정
	- x : 텍스트 삭제후 일반 모드 이동
	- y : yank, 복사하기후 일반 모드 이동
	- p : paste, 커서 다음 칸에 붙여넣기
	- P : 커서 이전 칸에 붙여넣기
	- V : 줄 단위로 블록 지정가능한 비주얼모드로 이동
	- d : 잘라내기, 삭제후 붙여넣기 가능
